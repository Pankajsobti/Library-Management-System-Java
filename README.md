# Library Management System (Java)

A console-based Library Management System developed using Java and Object-Oriented Programming concepts.

## Features

- Add new books
- View available books
- Issue books
- Return books
- Custom exception handling
- Reminder thread for due books

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA

## Project Structure

src  
├── model  
│   └── Book.java  
├── service  
│   └── LibraryService.java  
├── exception  
│   └── BookNotFoundException.java  
├── threads  
│   └── ReminderThread.java  
└── Main.java  

## How to Run

1. Clone the repository

git clone https://github.com/Pankajsobti/Library-Management-System-Java.git

2. Open the project in IntelliJ IDEA

3. Run `Main.java`
